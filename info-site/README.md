## Usage

### Local Development
Simply open `index.html` in a browser:
```bash
# From project root
open index.html
# or
firefox index.html
# or
chrome index.html
```

### With Local Server
For best results, serve with a local server:
```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server

# PHP
php -S localhost:8000
```

Then navigate to: `http://localhost:8000`
